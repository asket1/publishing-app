Project is running in Docker containers on VPS

MERN project:    
 Backend: Node.js, Express.js, MongoDB, JWT, Bcrypt  
 Frontend: React.js, Redux

To start server:  
 cd backend  
 npm install  
 npm start

To start client:  
 cd frontend  
 npm install  
 npm start
